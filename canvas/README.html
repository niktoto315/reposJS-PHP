Немного информации о проекте

При создании проекта были учтены следующие требования:
  • Создайте игру из урока
  • Добавьте стартовый экран:
  • название игры,
  • инструкцию к игре,
  • выбор космического корабля игрока,
  • кнопку "Начать".
  • Добавьте игровой экран:
  • добавьте количество жизней,
  • количество очков,
  • время игры (таймер обратного отчета),
  • кнопку "Пауза".
  • Изменения по возможности:
  • уровни в игре.
Остальные изменения добавлять откровенно лень, ведь дальше с это игрой ничего не произойдёт.
